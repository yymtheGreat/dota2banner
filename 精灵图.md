# CSS其他部分

## 元素隐藏

```css
☞  超出父元素部分隐藏
	overflow:hidden;   [塌陷,清除浮动]


☞ 具体属性:
			/* 默认: 如果父元素中的内容超出父元素后,没有滚动条,那么超出的部分也会正常显示 */
			/* overflow: visible; */

			/* 可以将超出父元素的部分隐藏 */
			/* overflow: hidden; */


			/* 给父元素设置滚动条 */
			/* overflow: scroll; */

			/* 根据内容判断是否要添加滚动条, */
			/* 如果内容的高度大于父元素的高度或者宽度,添加滚动条,反之不添加 */
			overflow: auto;



☞ 元素整体隐藏
	 1:  display: none;   隐藏元素, 不占位置
		 display: block;  显示元素


	 2:  visibility: hidden;  隐藏元素会占位置(清除浮动会用)
```
